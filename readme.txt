========================================================================
＊紹介

Netscape 7/8とMozillaに装備されているSidebarを別ウィンドウで表示させられ
るようにする拡張機能です。

以下の操作で、元のウィンドウからSidebarを分離できるようになります。

・Sidebar 上でコンテキストメニューを開き、「 Sidebar を分離」を選択する
・Sidebar の「 Sidebar を分離」ボタンをクリックする
・Sidebar のタイトル部分を中ボタンクリック（ Ctrl ＋左クリック, Mac では
  Command ＋クリック）する
・Sidebar のタイトル部分をドラッグする

元に戻す時は、逆の操作をして下さい。

また、この拡張機能の導入後は、Sidebarパネルについて以下の操作が可
能になります。

・ドラッグ＆ドロップでのパネルの並べ替え
・ドラッグ＆ドロップでのパネルの追加（リンク・ブックマークなど）
・パネルの削除（パネルをCtrl(Command)+ドラッグ）
・パネルのリネーム

step1: ファイルのコピー
  sidebar.xpiをブラウザのウィンドウにドロップすると、インストー
  ルが開始されます。
  途中、日本語の言語パックを登録するかどうかの確認があるので、必要に応じ
  て選択して下さい。

step2: NS/Mozの再起動
  ファイルのコピーが終わったら、NS/Mozを再起動します。
  コピーしたファイル群が、起動時にNS/Mozに登録されます。

＊アンインストールの手順

サイドバーのコンテキストメニューから「Version Info（バージョン情報）」
を選択し、表示されたダイアログウィンドウで、「Uninstall（アンインストー
ル）」と書かれたボタンをクリックして下さい。自動でアンインストールが行
われます。

なお、ファイルの削除は自動では行われません。表示されるメッセージに従っ
て、ブラウザを終了させた後に、指定されたファイルを手動で削除して下さい。


ブラウザ自体起動できないなどの緊急の場合には、以下の手順に従って、手動で
アンインストールを行って下さい。

  1. NS/Moz を終了させる。簡易起動（高速起動）が有効になっている場合、
     タスクトレイのアイコンを右クリックして、そちらも終了させる。
  2. ユーザープロファイルディレクトリか、 NS/Moz をインストールした
     ディレクトリ（管理者権限でインストールした場合）の中にある
     /chrome/ フォルダから、 sidebar.jar, overlayinfo ディレクトリ,
     chrome.rdf を削除する。
  3. 管理者権限でインストールした場合、installed-chrome.txt をメモ帳等
     で開き、以下の行を削除する。
     ・content,install,url,jar:resource:/chrome/sidebar.jar!
                           /content/sidebar/
     ・content,install,url,jar:resource:/chrome/sidebar.jar!
                           /content/sidebarWindowNavigator/
     ・locale,install,url,jar:resource:/chrome/sidebar.jar!
                           /locale/en-US/sidebar/
    (・locale,install,url,jar:resource:/chrome/sidebar.jar!
                           /locale/ja-JP/sidebar/)

この手順では設定情報などが残ったままになりますが、ブラウザを使用する上
では問題ありません。
